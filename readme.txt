The easiest way to add Socicon, the social icon font, to your site!

== Description ==

Socicon is a lightweight font, dedicated to social icons.

If you've ever used FontAwesome just for the social icons, you now have an easy to use alternative!

== Installation ==

1. Upload the socicon folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
